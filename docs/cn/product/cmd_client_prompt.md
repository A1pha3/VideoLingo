# 批量翻译

- VideoLingo的系统中是否有命令行工具，可以完成下面这个功能：
  1. 命令行接受两个参数，一个是输入的视频路径，一个是输出的视频路径
  2. 读取输入视频路径中的视频，把没有翻译成功的视频，逐个按顺序：翻译并生成字幕，配音。
  3. 把最终配音成功的视频文件放到输出视频路径中，并记录哪些文件翻译并配音成功。

- 如果没有能实现这个功能的client，考虑如何优雅，简洁的复用现有系统代码，详细设计一个能够完成这个功能的方案，并记录到batch_transalte.md，放到docs/cn/product 目录中。
